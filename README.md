# Windows-Event-Log-Analysis

## Objective 
Analyze Windows Event Logs using Event Viewer to identify successful logins, faild logines, and DriverFramework events. 

## Tools used 
- Windows Event Viewer

## Events Investigated
- Event ID 4624 - Successful logins
- Event ID 4625 - Failed logon
- Event ID 2100 - DriverFrameworks
- Event ID 2101 - DriverFrameworks

## Finding 
- Verifide successful login events
- Identified faild login attempts 
- Analyzed DriverFrameworks operational logs
- Recorded timestamps and event details

## Skils Demonstrqated 
- Windows Log Analysis
- Event Investigation
- Security Monitoring
- Digital Forensics Basics

## Conclusion 
This project Demonstrated basic Windows Event Log analysis using Event Viewer for cybersecurity and digital forensics
